# intelligent-meeting-minutes-assistant

智能会议纪要生成助手MVP核心实现

项目经历：我曾独立负责“智能会议纪要生成助手”小程序从0到1的MVP开发。项目旨在解决B端会议记录低效痛点。我使用Cursor进行核心开发，结合大模型API实现语音转文本及要点自动总结。通过设计用户故事与交互流程，我快速完成了前端界面与后端逻辑开发。上线后内部测试显示，纪要整理时间平均缩短65%，用户满意度达4.8/5。该项目让我完整实践了AI产品从需求分析到代码落地的全流程。

## 运行
```bash
pip install -r requirements.txt
python main.py
```
